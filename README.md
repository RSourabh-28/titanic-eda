# titanic-eda

# Titanic EDA Project

## OVERVIEW
Exploratory Data Analysis on the titanic Dataset

## Key Findings

- Only 38.38% of passengers survived
- 74% of females survived vs 19% of males
- 1st class had 63% survival vs 24% for 3rd class
- Higher fare strongly correlated with survival

## Tools Used

- Python
- Pandas
- Matplotlib
- Seaborn
